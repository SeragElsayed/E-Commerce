# E-Commerce
E-commerce web application where users can add items to their shopping cart and make orders, Admin can 
add products and accepts orders.

###Used Technology.
- ASP.NET MVC.
- Bootstrap.
- JQuery.
- MS SQL Server.
- Login is managed by session.


###Database
	Import EcommDB to your SQL server.
